# Бот для клуба риичи "Ламповый север"

Этот бот пока что мало что умеет и был написан for fun.
Умеет он, собсвтенно, показывать статистику, взятую с сайта клуба.


## Установка и запуск

```
pip install -r requirements.txt
python3 main.py
```
